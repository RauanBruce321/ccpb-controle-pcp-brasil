CCPB - CONTROLE DE PCP BRASIL
PROJETO ORGANIZADO PARA GITHUB PAGES

ESTRUTURA
- index.html: estrutura das telas
- css/style.css: estilo visual completo
- js/app.js: funcionamento atual completo
- js/firebase-config.js: configuração preparada para a próxima etapa
- assets/: logo e ícones
- manifest.json: identificação do aplicativo

IMPORTANTE
Esta organização preserva o comportamento da versão publicada.
O Firebase ainda não foi ativado no funcionamento do sistema nesta entrega;
o arquivo de configuração foi separado e deixado pronto para a conexão posterior.

COMO TESTAR NO COMPUTADOR
Abra o index.html. Para testar recursos web com mais fidelidade, use Live Server.

COMO ATUALIZAR O GITHUB
1. Apague/substitua os arquivos antigos do repositório pela estrutura desta pasta.
2. No GitHub Desktop, faça Commit.
3. Clique em Push origin.
4. O GitHub Pages atualizará automaticamente.

NÃO ENVIE A PASTA EXTERNA PARA DENTRO DO REPOSITÓRIO.
Envie o conteúdo dela, deixando index.html na raiz do repositório.
